* 10.0.0.55_login

** 作用
   用 Python 登入/登出 10.0.0.55

** 用法
   在脚本目录下新建 config.json, 格式如下

   #+begin_src json :results output
     {
         "username": "1120172333",
         "password": "Th1s_n0T_pW0"
     }
   #+end_src

   然后执行
   =python 10_0_0_55.py login= 登录,
   =python 10_0_0_55.py logout= 注销.

** 依赖
   - python 3.6+
   - requests
